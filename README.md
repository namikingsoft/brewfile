Brewfile
==============================
My Brewfile on OSX.


Getting Started
------------------------------
Input the following line after [install Homebrew](http://brew.sh/index_ja.html).

```sh
brew tap rcmdnk/file
brew install brew-file
brew file set_repo -r namikingsoft/brewfile
brew file update
```
